## How to Use

```shell script
mvn archetype:generate \
    -DarchetypeCatalog=remote \
    -DarchetypeArtifactId=quark-archetype \
    -DarchetypeVersion=0.0.1-SNAPSHOT \
    -DarchetypeGroupId=com.dream \
    -DgroupId=com.example \
    -DartifactId=demo \
    -Dpackage=jar \
    -Dversion=0.0.1-SNAPSHOT
```
